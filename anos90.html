Anos 90

Nos anos 90 faltava somente uma coisa, uma maneira facil e rapida de se navegar pela internet, com isso em mente tim bernes-lee desnvolveu o WWW junto disso veio

  • HTML
  • HTTP
  • URL

Porém um problema ainda se continuava a lentidão da internet discada

Anos 2000

O problema dos anos anteiors veio a ser resolvido nos anos 2000 com o surgimento WI-FI baseado em padroes IEEE 802.11 eliminou a necessidades de cabos, isso basicamente moldou a internet atualmente porém manter servidores era caro faltava algo

Anos 2010

Pra resolver o problema acima nos anos 2010 surgiu as NUVENS servidores onlines que permite a utilização de softwares servidores armazenamento e banco de dados sem ter eles fisicamente, os principais são

  1. Amazon web service
  2. MIcrosoft Azure
  3. Google cloud

com isso datas centers começou a se popularizar e ficar mais comum
